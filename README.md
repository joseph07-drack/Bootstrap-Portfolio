# Portofolio website Using Bootstrap

> Moving from Setting up a mobile version skeleton to a desktop version skeleton version of a Portofolio website with a refernce to a figma UI using Bootstrap

## Built With

- HTML and CSS
- Bootstrap
- npm, npx,Linters

## Live Domo

[Live demo](https://joseph07-drack.github.io/Bootstrap-Portfolio/)

## Getting Started

To get a local copy up and running follow these simple example steps.

- Having the basics understanding of **HTML,CSS, Git, GitHub and GitHub Flow,Linters**

- Installing Bootstrap on your local machine

- Using the git command _git clone git@github.com:joseph07-drack/Bootstrap-Portfolio.git_ to clone this project

- Changing the directory to **Bootstrap-Portfolio** folder to get all files

- Running the _npm install_ to get all the modules from package.json

- Use _"npx hint ."_ and _"npx stylelint /\**/*.{css,scss}""_ to check for possible errors within the project
